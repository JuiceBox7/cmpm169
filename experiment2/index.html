Experiment 2 - Vector Art, Animation & Interactivity

Imitate

For this assignment, I looked at the references on the Generative Design website and took two as inspiration. The first one I took some inspiration from was a piece that traced a semi transparent ring across a blank white canvas, and the semi transparent black stroke color allowed it to essentially shade the page. I thought the shading effect to be really cool, and decided to take this piece of code from it to incorporate into my experiment. As for the second piece I used as inspiration, it was a piece that mirrored what was drawn with the mouse in 8 directions stemming from the center.

Integrate

For integration, I put these ideas and pieces together to make a confusing/unique way of drawing on a canvas, along with it being semi transparent so that it can have the shading effect. The “stroke(0, 50);” line is how the stroke becomes semi transparent, and everything else is to edit the background, cursor, and width of the stroke. I then use some math to have fun with the lines that are drawn in the draw() function. The math essentially mirrors the mouse's x and y coordinates between the negative, negative quadrant and positive, positive quadrant of the canvas.

Innovate

For the innovation part of the process, I used my JavaScript knowledge to give the canvas an event listener for click, as simply checking to see if the mouse is clicked in the draw function can lead to errors, since it's called per frame. This allowed for it to be more user friendly, as it just reads the click, and waits until the user completes the click (releasing left-click) to fire the event. Instead of calling the draw function from the event listener, I simply change a boolean variable which is checked in the draw function, to call two functions, as well as draw the lines. The UpdateMousePos() function updates the x,y variables to be the mouse's coordinates, and updateNum() decreases the value of num up to 0.5, then “resets it” to get some interesting line results.
